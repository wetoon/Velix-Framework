ğŸš€ Getting Started

  1. Place your built frontend files (React, Vue, Svelte, etc.) inside public/.
  2. Define your backend routes in public/app.php.
  3. Velix will automatically handle routes and serve your frontend seamlessly.

ğŸ§© Example

<?php
require __DIR__ . '/../Velix.php';

$app = new Velix();

$app->get('hello/{name}', function(Request $req, Response $res, $name) {
    return [ 'message' => "Hello, $name!" ];
});

$app->post('user', function(Request $req, Response $res) {
    $username = $req->input('username');
    return [ 'created' => $username ];
});

$app->dispatch();
